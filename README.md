\# VaultAI



VaultAI is an AI assistant that reads from a \*\*FAISS index of your Obsidian Vault\*\* and answers questions based on your personal notes. It allows you to query your knowledge base in natural language and get contextual, accurate responses derived directly from your vault.



\## Features



\- 📚 \*\*Obsidian Vault Integration\*\*:  

&#x20; VaultAI loads a FAISS index built from your Obsidian notes, so it can answer questions using semantic search over your personal knowledge.



\- 🌐 \*\*Integration with thehelpers\*\*:  

&#x20; The project also integrates with the website \*\*\[thehelpers](https://thehelpers.vercel.app)\*\*, expanding its capabilities by connecting to external tools or systems via that platform.



\## Usage



\- Train or update the FAISS index with your Obsidian notes.

\- Run the appropriate Python script to start the VaultAI interface.

\- Ask natural‑language questions, and VaultAI will search your vault and return relevant answers.



\## Script Versions



\- ✅ `vault\_ai\_gui\_claude.py` — \*\*Final version\*\* of the GUI application.  

&#x20; This is the most refined and fully debugged version, optimized for interactive use.



> ⚠️ \*\*Note\*\*: While this project was \*\*debugged using Claude\*\*, the \*\*entire code was not written by Claude\*\*. Earlier versions of the code (`vault\_ai\_gui.py`, `vault\_ai\_gui\_GPT.py`, etc.) were debugged with their respective models (e.g., GPT, Claude).  

> This was done as a kind of \*\*experiment/challenge\*\* to see which model produced the best‑debugged, most stable version.  

> The final version (`vault\_ai\_gui\_claude.py`) is recommended for regular use.
